# Card Matching Memory Game

Acesta este un joc de tip card matching (memory game) dezvoltat în Processing, realizat ca proiect universitar. Scopul jocului este de a găsi toate perechile de cărți identice cu un număr cât mai mic de încercări și într-un timp cât mai scurt.

---

## Tehnologii utilizate
- **Limbaj:** Java (Processing)
- **Framework:** Processing

---

## Funcționalități principale
- Sistem de întoarcere a cărților la click
- Detectarea perechilor corecte
- Ascunderea automată a cărților care nu se potrivesc
- Sistem de scor sau număr de mutări
- Interfață grafică interactivă
- Sistem de finalizare a jocului (win condition)

---

## Ce am învățat
- Dezvoltarea aplicațiilor grafice folosind Processing
- Gestionarea input-ului de la mouse
- Implementarea logicii unui joc bazat pe stări
- Organizarea codului folosind concepte OOP
- Gestionarea evenimentelor și a randării grafice
- Crearea unei interfețe interactive pentru utilizator
